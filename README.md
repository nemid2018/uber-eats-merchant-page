# Uber Eats for Merchants

A marketing and merchant onboarding site for Uber Eats, built with React + TypeScript + Vite.

## Dev

```bash
npm install
npm run dev
# → http://localhost:8080/uber-eats-merchant-page/
```

## Build

```bash
npm run build
```

## Deploy

Pushes to `main` automatically deploy to GitHub Pages via `.github/workflows/deploy.yml`.

Live site: https://nemid2018.github.io/uber-eats-merchant-page/

## Stack

- Vite + React + TypeScript
- Tailwind CSS
- Framer Motion
- React Router v6
